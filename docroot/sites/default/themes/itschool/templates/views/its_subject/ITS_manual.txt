Курс (общий курс):
- Content -> Add content -> Курсы
-- Title - название курса
-- Language - язык
-- DOMAIN ACCESS OPTIONS - выбор к какому городу отностится курс
--- Send to all affiliates - добавить ко всем городам (если включен, перебивает выбор ниже)
--- Publish to - выбор города (IT School - можно игнорировать, будет спрятом позже)
-- FRONT PAGE
--- Icon - иконка курса для главной страницы и страницы курсов. Можно как загрузить новую картинку, так и выбрать существующую на вкладке Library.
--- Short Description - краткое описание курса (4-6 слов) для главной страницы и страницы курсов
--- Position - положение курса на главной странице, в левой (Left) или правой (Right) колонке
--- Weight - вес влияет на порядо отображения курсов, курсы с меньшим весом будут в начале
-- Body - детальное описание общего курса
-- Meta tags - мета теги для детальной страницы курса.
--- Page Title - мета заголовок, по умолчанию "название курса | имя сайта"
--- Description - мета описание
-- URL path setings - настройки урла страницы, по умолчанию генерирутся автоматически с префикса course и тайта (course/[название_курса]). Выключите "Generate automatic URL alias" что бы ввести урл вручную

Предмет (детальный курс):
- Content -> Add content -> Предмет
-- Title
-- Language
-- DOMAIN ACCESS OPTIONS
-- Class - общий Курс к которому относится предмет
-- START DATE - дата начала курса, может иметь несколь значений, "Add anothe item" - чтобы добавить вторую и более дат
-- Duration - продолжительность курса в часах, наприме 46.
-- Frequency - Интенсивность курса, наприме "3 раза в неделю"
-- Body - детальное описание
-- Преподаватель - Преподаватель курса, начните вводить имя что бы появились варианты выбора.
-- Price - цена курса в гривнах

Преподаватель:
- People -> Add user
-- Full Name - фамилия и имя преподавателя
-- Position - должность
-- Image - фото
-- Description - описание преподавателя (опыт, специальность, т.п.)
-- Username - имя пользователя для логина (обычно = Full Name). На странице создание предмета, преподаватель будет искаться по значению этого поля.
-- E-mail address
-- Password
-- Confirm password
-- Status - выбрать Active
-- Assignable roles - выбрать роль Teacher
Остально оставить по умолчанию
